Importar / Exportar Datos

¿Cómo funciona?

Este módulo te permite respaldar y restaurar todos los datos de tu tienda (productos, categorías, contenido, redes sociales) mediante un archivo JSON.

Se exporta:

  • • Productos (título, precio, descripción, SKU, estado)
  • • Categorías
  • • Contenido (Nosotros, Ubicación, páginas legales)
  • • Redes sociales

No se exporta:

  • • Imágenes (se quedan en Cloudinary)
  • • Configuración de Firebase
  • • Credenciales de admin

Las URLs de las imágenes se incluyen pero las imágenes físicas no.
